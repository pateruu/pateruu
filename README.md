## 🚀 Projects

- **[Calorie Tracker PCB](https://github.com/pateruu/Calorie_Tracker_PCB)**  
  Handheld ESP32-S3 calorie tracker with a custom PCB, OLED display, button-based input, USB-C power/programming, battery charging, and onboard 3.3V regulation.

- **[LEDify](https://github.com/pateruu/Ledify)**  
  ESP32/ESP8266-based smart LED controller with embedded firmware and a responsive web UI for real-time color and animation control.

- **[Biro-1 Media Controller](https://github.com/PROJECT-BIRO/Biro-1)** *(ongoing)*  
  Custom PCB media controller with mechanical keys, a rotary dial, Bluetooth + USB-C connectivity, and firmware integration.

- **[MVM Accelerator](https://github.com/pateruu/mvm-accelerator)**  
  FPGA-based matrix-vector multiplication engine built with SystemVerilog and Vivado, focused on high-throughput datapath design and hardware optimization.  

- **[Fixed-Point Tanh Accelerator](https://github.com/pateruu/fixed-point-tanh-accelerator)**  
  23-stage pipelined fixed-point tanh accelerator in hardware, targeting efficient arithmetic implementation and timing closure on FPGA.
